# ZSTUN Probing Analysis

## Purpose
This module analyzes the probing outputs generated by: `zstun_probing`. The goal of this module is to identify behavioral deviations between:

- P1 probe (vanilla STUN traffic)
- P2 probe (replayed VoIP STUN traffic)
- P3 probe (modified VoIP STUN variants)

The module detects IPs that behave differently under different STUN packet types and classifies them into multiple behavioral categories.

## Installation
- `pip install pandas matplotlib` required to make plots has to be done for the script to run.

## Supported Deviation Types
The framework currently supports three major deviation categories:
- UDP Deviation
- ICMP Deviation
- No-Response Deviation
